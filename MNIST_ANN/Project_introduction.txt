# MNIST Classification with ANN

이 프로젝트는 딥러닝 프레임워크(PyTorch, TensorFlow 등)의 자동 미분 기능을 사용하지 않고, 순수 Python과 CuPy를 사용하여 인공신경망(Artificial Neural Network)을 밑바닥부터 구현한 프로젝트입니다.

MNIST 손글씨 데이터셋을 학습하여 0부터 9까지의 숫자를 분류하며, CuPy를 활용한 GPU 가속을 통해 빠른 학습 속도를 지원합니다.

* **Custom Implementation**: `Forward`, `Backward` (Backpropagation), `Optimizer` 등 딥러닝의 핵심 로직을 직접 수식으로 구현했습니다.
* **GPU Acceleration**: `Numpy` 대신 `CuPy`를 사용하여 대량의 행렬 연산을 GPU에서 처리합니다.
* **Standard Architecture**: Input(784) -> Hidden(128) -> Hidden(64) -> Output(10) 구조의 3층 신경망입니다.
* **High Performance**: 간단한 구조임에도 MNIST 데이터셋에 대해 약 97% 이상의 정확도를 달성했습니다.

## Project Structure

최상위 폴더
├── ver_pytorch.py             # 파이토치 버전
└── My_ver
        ├── ANN_Model.py       # 신경망 클래스 (초기화, 순전파, 역전파, 손실함수 등 포함)
        ├── train.py           # 모델 학습 스크립트 (데이터 로드, 학습 루프, 모델 저장)
        ├── run.py             # 추론(Inference) 스크립트 (저장된 모델 로드, 테스트 및 시각화)
        ├── model.pkl          # 학습 완료된 가중치 파일 (git에는 포함되지 않음)
        ├── data/              # MNIST 데이터셋 저장 폴더
        └── .gitignore         # git 제외 파일 설정

    
# Requirements
이 프로젝트를 실행하기 위해서는 아래의 라이브러리가 필요합니다.

Python 3.x

CuPy (NVIDIA GPU 및 CUDA 필요)

PyTorch, Torchvision (데이터셋 다운로드 용도)

Matplotlib (결과 시각화)

Tqdm (진행바 표시)

# 사용법

1. 학습 (Training)

    모델을 학습시키고 가중치 파일(model.pkl)을 생성합니다.

    cd MNIST_ANN/My_ver

    python3 train.py

2. 실행 및 테스트 (Inference)

    python3 run.py

    # 만약 본인의 데이터 넣고싶으시면 코드 수정하시면 됩니다. 저 편하려고 그냥 데이터 불러온거에요
